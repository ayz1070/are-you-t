# 🌱 너티야 (Neottya)

> **MBTI 기반 커뮤니티 앱**  
너티야는 MBTI 성격 유형을 기반으로 소통하는 커뮤니티 플랫폼입니다.  
비슷한 성향을 가진 사람들과 공감하고, 의견을 나누고, 친구가 되어보기도 하며 반대 성향을 가진 사람들에게 의견을 듣고, 이해하는 커뮤니티 입니다.

---

## ✨ 주요 기능

- 🔍 **게시판**  
  MBTI별 자유로운 글 작성 및 댓글 기능

- 💬 **댓글 및 대댓글**  
  사용자 간 실시간 소통 가능

- ❤️ **좋아요**  
  공감할 수 있는 게시글과 댓글에 좋아요 기능 제공

- 🔐 **회원가입 및 로그인**  
  이메일 및 소셜 로그인 지원 (예정)

- 🧑‍💼 **프로필 관리**  
  MBTI, 닉네임, 프로필 이미지 설정 가능

---

## 🧩 사용 기술

### 🖼 Frontend

| 기술 | 설명 |
|------|------|
| Flutter | 크로스 플랫폼 UI 프레임워크 |
| Riverpod | 상태 관리 라이브러리 |
| GoRouter | 화면 전환 및 라우팅 처리 |
| Dio | 네트워크 통신 라이브러리 |
| json_serializable | JSON 직렬화 자동화 |

### ⚙ Backend

| 기술 | 설명 |
|------|------|
| Spring Boot | 백엔드 서버 프레임워크 |
| Spring Data JPA | ORM 및 데이터베이스 연동 |
| MySQL | 관계형 데이터베이스 |
| Swagger | API 문서 자동화 |
| Lombok | 코드 간결화를 위한 어노테이션 |
